⛅️ GitHub Gist 同步

使用 GitHub Gist 在裝置間同步筆記。需要 GitHub 帳號與個人 Access Token（repo/gist 權限）。

1. 前往 github.com → Settings → Developer settings → Personal access tokens
2. 建立 token，只選 gist 權限
3. 在 gist.github.com 新建一個 Gist，複製網址中的 ID
Token 僅存於您的本機，不會上傳至任何伺服器

🤖 設定 AI（OpenRouter）

KLaws 使用 OpenRouter 提供的免費 AI 模型。
註冊免費帳號後取得 API Key 即可使用，每天最多 200 次，不需信用卡。

1. 前往 openrouter.ai 免費註冊
2. 右上角頭像 → Keys → Create Key
3. 複製 sk-or-v1-... 貼到下方
Key 僅存於您的本機，不會上傳至任何伺服器
